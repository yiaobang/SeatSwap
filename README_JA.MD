# 🎲 SeatSwap - 多言語対応・スマート席替えシステム
[🇨🇳 简体中文](./README.MD) | [🇯🇵 日本語](./README_JA.MD) | [🇺🇸 English](./README_EN.MD)

**SeatSwap** へようこそ！このアプリは、学校の先生やイベント主催者のための「自動席替え・座席抽選」ツールです。
「クラス全員を完全にランダムで席替えさせたい」「配置が難しい留学生や外国人の名前もきれいに表示したい」というお悩みを、ボタン一つで解決します！

---

## ✨ 主な特徴

* **圧倒的な公平性**：オンライン銀行などで使われる高度な暗号化ランダムアルゴリズムを搭載。完全に公平な席替えを実現します。
* **柔軟な特別ルール**：特定の生徒を「1行目（最前列）」や「3列目」に固定し、残りの座席を他の生徒でランダムに埋めることができます。
* **🌍 留学生・外国人名対応（自動改行機能）**：カタカナ名とアルファベット名が混ざった長い名前でも、かっこ `()` を使うことで（例：`田中(Alex)` ）、アプリが座席内で**自動的に上下2行に改行して美しく整列**させます。レイアウトが崩れる心配はありません！
* **ワンクリック言語切り替え**：**日本語・中国語・英語**に完全対応。いつでもデザインを崩さずに切り替えられます。

---

## 🛠️ 事前準備（初心者向け）

席替えを始める前に、パソコンの「メモ帳」などを使って、2つの `.txt` ファイルを用意してください。

### 1. 生徒の全員名簿（例：`students.txt`）
1行に1人の名前を記入してください。
> **💡 外国人名・ニックネームのコツ：**
> 英語名や本名をかっこ `()` で囲んで追記すると、座席表で上下にきれいに並びます。
>
> **書き方見本：**
> 佐藤
> 鈴木(Sam)
> Michael(マイケル)
> 田中

### 2. 特別席ルール名簿（例：`rules.txt` ※特別な希望者がいない場合は不要です）
* 特定の**行（横の並び）**に固定したい場合：`R` + `行番号` + `.` + `名前`
* 特定の**列（縦の並び）**に固定したい場合：`C` + `列番号` + `.` + `名前`
> **⚠️ 注意：かっこ付きの名前の場合、ルールファイル内の名前も全員名簿と完全に一致させてください。**
>
> **書き方見本：**
> R1.佐藤  *(佐藤さんを必ず 1 行目のどこかに配置する)*
> C3.鈴木(Sam)  *(かっこ付きの鈴木さんを必ず 3 列目のどこかに配置する)*

## 🚀 使い方（操作手順）

1.  **アプリを開く**：デスクトップまたはフォルダからアプリを起動します。
2.  **言語を選ぶ**：左上の `🌐 Language` から `日本語` を選択します。
3.  **座席の数を決める**：`行数` と `列数` のマスに、教室の座席数を入力します（例：5行・6列＝30席）。
4.  **ファイルを読み込む**：
    * **[全体名簿をインポート]** ボタンを押し、用意した `students.txt` を選びます。
    * **[特別席ルールをインポート]** ボタンを押し、用意した `rules.txt` を選びます（ない場合はスキップ）。
5.  **席替えを実行**：一番下にある大きな **[🎲 席替えを実行する]** ボタンを押します。
6.  **結果を確認**：画面右側に座席表が作られます。特別ルールが適用された生徒には `★` マークが表示され、かっこ付きの国際的な名前は自動で上下2行に美しく表示されます。
