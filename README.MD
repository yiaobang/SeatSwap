
# 🎲 SeatSwap - 智能多语言随机排座系统

[🇨🇳 简体中文](./README.MD) | [🇯🇵 日本語](./README_JA.MD) | [🇺🇸 English](./README_EN.MD)

欢迎使用 **SeatSwap**！这是一个专门为老师、会议组织者打造的“抽签排座位”神器。
不管你是想让全班同学完全随机换座位，还是有些同学因为特殊原因（比如近视、听力不好、个子矮）必须固定在某一排或某一列，这个软件都能一键帮你搞定！

---

## ✨ 核心亮点

* **真正的公平**：内置银行加密级别的随机算法，绝对没有“暗箱操作”，比普通的摇号更随机、更公平。
* **聪明的人性化规则**：支持给特殊学生“留座”（例如：指定张三必须坐第1排，或者李四必须坐第3列），其余空位由其他同学随机填补。
* **🌍 完美的留学生支持（双语姓名自动换行）**：如果班里有留学生或者外国友人，你可以在名单里写上他们的英文名，用括号包裹，例如 `王小明(Leo)`。软件在排座时会**自动换行精美排版**，既能同时看懂中英文，又绝不会把座位格子挤变形！
* **一键换语言**：支持**中文、日文、英文**三语一键切换，界面丝滑好看。
* **讲台视角**：排座结果直接以“面向讲台”的直观表格展示，一目了然。

---

## 🛠️ 准备工作（小白必看）

在开始摇号前，你需要用电脑自带的“记事本”准备两个 `.txt` 格式的文本文件。

### 1. 准备总学生名单（比如命名为 `students.txt`）
里面每行写一个学生的名字，不要有空格。
> **💡 留学生/双语名字小技巧：**
> 如果有外国名字或英文昵称，请用英文括号 `()` 拼在名字后面，软件会自动帮它们换行对齐哦！
>
> **正确示范：**
> 张三
> 李四(Sam)
> Michael(麦克)
> 王五

### 2. 准备特殊座位规则（比如命名为 `rules.txt`，如果没有特殊要求的学生，可以不准备）
* 如果想让某人**固定在某一排**：用 `R` + `行号` + `.` + `名字`。
* 如果想让某人**固定在某一列**：用 `C` + `列号` + `.` + `名字`。
> **⚠️ 注意：如果名字带括号，规则文件里的名字必须和总名单里的一模一样！**
>
> **正确示范：**
> R1.张三  *(表示张三必须坐在第 1 排的某个位置)*
> C3.李四(Sam)  *(表示带英文名的李四必须坐在第 3 列的某个位置)*

## 🚀 怎么使用？（保姆级操作指南）

1.  **双击打开软件**：你会看到一个非常清爽的界面。
2.  **选择你的语言**：在左上角 `🌐 Language` 下拉框里选择 `中文`。
3.  **设置座位有多少行和列**：在 `行数` 和 `列数` 输入框里，输入教室里的座位排数和列数（例如：5行6列，就是30个座位）。
4.  **导入名单**：
    * 点击 **[导入总学生名单]** 按钮，选择你刚刚准备好的 `students.txt`。
    * 点击 **[导入特殊座位规则]** 按钮，选择你准备好的 `rules.txt`（如果没有特殊要求，这一步可以跳过）。
5.  **开始摇号**：点击最下方大大的 **[🎲 开始随机排座]** 按钮。
6.  **查看结果**：右侧会瞬间生成好看的座位表。名字后面带 `★` 的，说明他是享受了特殊照顾的学生；带括号的双语名字会被优雅地分为上下两行显示。

⚠️ **注意：如果特殊规则之间发生了冲突（比如大家都争着坐第1排，座位不够了），软件会弹窗贴心地提示你哪些同学没能安排进去，并自动把他们放到其他空位上，绝对不会让程序卡死或崩溃！**
